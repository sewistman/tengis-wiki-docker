# About

- [Docker](https://docker.com/) is an open source project to pack, ship and run any Linux application in a lighter weight, faster container than a traditional virtual machine.

- Docker makes it much easier to deploy [a Tengis Wiki server](https://github.com/haiwen/seafile) on your servers and keep it updated.

- The base image configures Tengis Wiki with the Tengis Wiki team's recommended optimal defaults.

If you are not familiar with docker commands, please refer to [docker documentation](https://docs.docker.com/engine/reference/commandline/cli/).
